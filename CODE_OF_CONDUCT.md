# Code of Conduct

## Our Pledge

We want Swiq to be a welcoming, respectful, and friendly project for everyone.

Everyone participating in this project is expected to treat others with respect regardless of experience level, identity, nationality, or background.

---

## Expected Behaviour

Examples of positive behaviour include:

- Being respectful.
- Giving constructive feedback.
- Helping new contributors.
- Being patient.
- Respecting differing opinions.

---

## Unacceptable Behaviour

Examples include:

- Harassment or discrimination.
- Personal attacks.
- Hate speech.
- Trolling.
- Threats or intimidation.
- Spam.
- Deliberately disrupting discussions.

---

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate this Code of Conduct.

Repeated or severe violations may result in a temporary or permanent ban from contributing.

---

## Reporting Issues

If someone is violating this Code of Conduct, please open a private issue or contact the project maintainer.

Reports will be reviewed fairly and respectfully.

---

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Project documentation
- Any official Swiq community spaces

---

Thank you for helping make Swiq a welcoming project for everyone.

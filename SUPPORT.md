# Support

Need help with Swiq?

## Before Asking

Please check:

- The README
- Existing GitHub Issues
- Existing Discussions

Your question may already have been answered.

---

## Getting Help

If you need help using Swiq, you can:

- Open a GitHub Discussion (preferred)
- Open an Issue if you've found a bug
- Contact the maintainer by email:

**bananacool467+1@gmail.com**

Please include:

- Your operating system
- Swiq version
- The code you're running (if applicable)
- The full error message
- Steps to reproduce the problem

The more information you provide, the easier it is to help.

---

## Bug Reports

If you've found a bug, please create a GitHub Issue instead of sending an email whenever possible.

Include:

- Expected behaviour
- Actual behaviour
- Steps to reproduce
- Screenshots or logs if relevant

---

## Feature Requests

Ideas are always welcome!

Please open an Issue explaining:

- What you'd like added.
- Why it would improve Swiq.
- Possible implementation ideas (optional).

---

Thanks for using Swiq!
